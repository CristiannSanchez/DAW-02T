¿Qué es Github?


Es un servicio muy útil para los desarrolladores, el cual consta de un servicio basado en la nube, ya que aloja un sistema de control de versiones (VCS) llamado Git. (Git es una herramienta que realiza una función del control de versiones de código de forma distribuida)

Ventajas de Github


Permite a los programadores facilitar el proceso adaptados a cambios en proyectos los cuales son compartidos entre más programadores, a la misma vez que mantienen un control sumamente detallado de todo el progreso.


Limitantes de Github

algunos casos puede haber ciertas limitaciones en lo relativo a la facilidad de uso y eficiencia de GitHub.
En ocasiones surgen complicaciones entre el programa cliente y la compañía cuando,
por ejemplo, un servidor privado opera como host para el código creado.
Otra de las razones que motivan la elección de alternativas a GitHub es el empleo de un VCS
diferente no soportado por GitHub. Hoy en día existen diversas alternativas a GitHub, pero
